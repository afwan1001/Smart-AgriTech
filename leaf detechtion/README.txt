# 🌱 Smart AgriTech Platform

## 📌 Overview
The Smart AgriTech Platform is an integrated solution designed to support farmers with modern technology.

This system combines plant disease detection using image processing and a full-featured e-commerce platform where farmers can purchase agricultural products such as fertilizers, tools, and smart farming equipment.

---

## 🚀 Features
🌿 AI-based plant disease detection  
📷 Upload leaf images for analysis  
💊 Get treatment suggestions for diseases  
🛒 E-commerce platform for agricultural products  
🔐 Secure user authentication system  
💳 Multiple payment options (UPI, Card, Cash on Delivery)  
📊 Product filtering and sorting (like e-commerce platforms)  

---

## 🏗️ Architecture
User → React Frontend → Flask Backend → ML Model → Database → E-commerce System

## ⚙️ Setup Instructions
Clone repository:
git clone https://github.com/your-username/smart-agritech-platform.git

Frontend setup:
cd frontend
npm install
npm start

Backend setup:
cd backend
pip install -r requirements.txt
python app.py

Open browser:
http://localhost:3000
